# cloud-trace-bucket-vue-2
Front-end for CloudTraceBucket

## Required ENV variables:

`VUE_APP_STORAGE_API=`

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
